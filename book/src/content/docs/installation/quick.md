---
title: "Quick start"
description: "Download and install the prepoly toolchain."
---

You can download and install prepoly toolchain by running the command below:

```bash
curl -L https://raw.githubusercontent.com/cordx56/prepoly/refs/heads/main/scripts/install.sh | sh
```

Note that you have to add `$HOME/.prepoly/bin` to `$PATH` to use `prepoly` command.
