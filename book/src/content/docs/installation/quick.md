---
title: "Quick start"
description: "Download and install the Brass toolchain."
---

You can download and install Brass toolchain by running the command below:

```bash
curl -L https://raw.githubusercontent.com/brass-cz/brass/refs/heads/main/scripts/install.sh | sh
```

Note that you have to add `$HOME/.brass/bin` to `$PATH` to use `brass` command.
